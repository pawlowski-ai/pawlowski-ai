## Selected Work

### [MendWise](https://github.com/pawlowski-ai/repairmate)

AI-powered mobile repair assistant built with **Expo, React Native, Firebase, Gemini, and RevenueCat**.

Users describe a household or car repair problem, optionally add a photo, receive a likely diagnosis, request an alternative diagnosis, and continue into step-by-step repair guidance.

**Signal:** AI workflow design, multimodal intake, safety-aware prompt instructions, Firebase-backed AI operations, product-minded QA.

**Signal:** prompt constraints, model behavior control, red-team mindset, LLM safety QA.
